Speech Recognition Project

##  Description

This project is a simple speech recognition system built باستخدام Python.
It allows the user to record audio from the microphone and converts it into text (Speech-to-Text).

The project demonstrates how to use speech recognition libraries to process audio input and extract meaningful text output.

---

##  Features

* Record audio using microphone
* Convert speech into text
* Simple and fast implementation
* Demonstrates basic Natural Language Processing (NLP) concept

---

## Technologies Used

* Python
* SpeechRecognition Library

> The SpeechRecognition library supports multiple engines like Google Speech API and others. ()

---

## How to Run

1. Install dependencies:

```bash
pip install SpeechRecognition
pip install pyaudio
```

2. Run the script:

```bash
python untitled26.py
```

3. Speak into your microphone and see the output text.

---

## How It Works

1. The program captures audio from the microphone.
2. It processes the audio signal.
3. It sends the audio to a speech recognition engine.
4. The engine returns the recognized text.
